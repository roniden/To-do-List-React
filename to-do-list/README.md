# To-Do List (Vite + TS)
This is a simple **to-do list project**, created with the goal of studying and practicing **TypeScript**. The application was developed using **Vite** as the bundler and **TypeScript** for static typing.

## Description
The main objective of this project is to learn how to work with **TypeScript** in a real development environment. During development, I explored concepts such as static typing, interfaces, and other TypeScript features.

## Technologies
- Vite
- React
- TypeScript
- Tailwind

## How to run the project
1. Clone the repository
2. Run npm install
3. Run npm run dev
